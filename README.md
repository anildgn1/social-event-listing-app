# Social Event Listing Application

An Android-based event management and discovery application developed using Java and Android Studio.

## Features
- Event listing and event detail pages
- Event filtering and sorting
- Location-based event features
- Google Maps API integration
- Firebase integration
- Room Database support
- User-focused mobile interface

## Technologies
- Java
- Android Studio
- Firebase
- Room Database
- Google Maps API
- XML

## Project Purpose
This project was developed as a graduation project to create a mobile application where users can discover, add, view, and manage social events.

## Status
This project is under development.
